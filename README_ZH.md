## 关于我 | “识别优先的研究型构建者”

我就读于南京邮电大学经济学院，当前关注经济学、实证识别、政策评估，以及 AI 在研究流程中的实际应用。

我希望做的不是“技术上能跑”的研究，而是“结构上也有说服力”的研究：问题要干净，设计要可信，机制要讲清，表达要经得起审视。

## 学术定位

- 南京邮电大学经济学院
- 聚焦经济学、政策评估与因果识别
- 关注经济学 x AI、文本数据与计算社会科学
- 希望把研究做成可复现、可阅读、可应对审稿的完整系统

## 研究理念

我不把数据清洗、模型估计、表格图形和论文表达当作彼此分离的任务。它们本来就是同一个研究系统的不同环节。

我最看重的几件事：

- 从值得识别的问题开始
- 优先选择能自我辩护的识别设计
- 稳健性检验是为了澄清结论，不是为了装饰结果
- 输出不仅要技术正确，也要让读者先看懂逻辑

## 研究关注

### 1. 政策评估

- 面向现实问题的应用实证研究
- 以识别为核心的研究设计
- 关注结果背后的作用机制

### 2. 因果识别

- 双重差分及相关设计
- 工具变量与经验研究可信度
- 稳健性、异质性与机制检验

### 3. 经济学 x AI

- 面向经济研究的文本数据方法
- 面向研究流程的智能工具
- 用计算方法提升测度与证据生产效率

## 前沿兴趣

我希望继续把研究边界拓展到：

- 计算社会科学
- AI 辅助实证研究流程
- 非结构化数据的经济测度
- 可复现的研究系统
- 技术内容的高质量可视化表达

## 工作顺序

```text
QUESTION -> DESIGN -> IDENTIFICATION -> ROBUSTNESS -> MECHANISM -> COMMUNICATION
```

这是我最信任的研究顺序。它能让工作保持诚实。

## 我的判断标准

> 好的研究不是回归结果的堆砌，而是一个干净的问题、一个可信的设计，以及在尖锐追问下依然站得住的结论。
